## 怎么验收（建议照这个顺序真机走一遍）

开发服务器已在 http://localhost:3000 重启（已清 `.next`）。

1. **内容安全（最重要）**：新建一棵树时，在"现状/岔路"里写一句带危机信号的话（如：我不想活了）→ 应弹出温和的关怀覆层 + 求助热线，且**不**立即生成预测；点"我没事，继续"才继续。普通输入完全照旧。再在"和未来的自己聊"里发一句危机信号 → 应得到关怀回复 + 资源，且不发给 AI。
2. **分享**：人生树页点「📤 分享」→ 下载一张自包含 SVG，含红线声明"可能的人生，不是预测的命运"，不含薪资等隐私字段。
3. **本周回顾**：仪表盘点「📅 本周回顾」→ 看到本周完成数/连续天数/该复盘的决定与目标/本周达成的里程碑；"去复盘"能打开既有复盘表。
4. **对话式调路**：进某条路的未来自我对话，聊几句 → 点「🔄 据我们聊的，重推这条路」→ 这条路带着你说的补充重新推演。
5. **助手建目标**：规划助手里输入一个方向或让它铺开几条路 → 点「🎯 设成目标」→ 在树上长出该长期目标的分支。
6. **预测质量**：随便生成/重推一条路，看 AI 故事是否更扎实（真实签证里程碑+年份、具体职级与总包数字、真实摩擦、因果承接、无套话）。**注意**：质量提升靠的是改强了 enrich 提示词，要真看到效果需 `.env.local` 里有 DeepSeek key；没 key 走本地兜底（兜底文案仍较模板化，见"留作后续"）。
7. 切英文，新文案均英文。
8. 旧 localStorage 数据升级不丢、不报错。
