# Product direction (Route A)

Positioning: a **life-direction / decision + prediction tool**, NOT a daily to-do app competing with Todoist/Motion. The differentiator is the loop: **predict possible futures → decide between paths → daily action → feasibility rises → milestone → talk to your future self.**

- The prediction "life tree" + talk-to-future-self + decision panel are the unique value. Daily execution (calendar/tasks/habits) is the supporting layer — keep it solid but don't over-invest in planner depth.
- **Honesty over hype**: predictions are "believable possibilities", never fortune-telling. Avoid false precision. "Feasibility %" means *how reachable this path is for this person* (start vs demands), shown as "约 X%" (rounded to 5, capped 95) + an "AI 粗估，非精确概率" disclaimer. It rises with the user's real progress, framed as "your effort +X%", not cosmic probability.
- **Content safety**: crisis-signal detection + help resources (`packages/core/src/safety.ts` + SafetyCare) must stay; conservative detection.
- Give the user blunt, non-sycophantic feedback — name risks and tradeoffs.
